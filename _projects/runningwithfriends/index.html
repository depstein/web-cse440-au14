Overview

Running is a great way to stay healthy and active, but can also be a dull and repetitive routine without consistent motivation. Running with others can be an effective way to keep each other accountable and encouraged. There are many people that prefer to run with others as well as challenges that make it difficult to find running companions such as coordinating schedules, managing complicated logistics, and finding people they can run in sync with.

To address these concerns, we propose Running With Friends, a mobile application that brings convenience, variety, and social value to the running experience. With features to send out mass invitations and to find SmartMatches, Running With Friends simplifies the process of planning a run and finding companions, allowing runners to focus on what’s important – staying active and having fun doing it.

Motivation

People are often more motivated to run consistently when they have friends with similar running experience to keep them accountable. This application addresses the challenge of finding these friends by replacing the conventional way of scheduling an event (in advance),with a way to find available friends in a moment’s notice.

Socialize

This application’s features serve to simplify the process of finding running companions. By making it easier to integrate existing and new friends into one’s running routine, this application helps one cultivate their own community of running companions.
